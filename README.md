# 📱Mobile Analyzer 2026
**An Intelligent Data-Driven Insights Platform for the Smartphone Market**

---

##  Project Overview
The **Mobile Analyzer 2026** is a high-end data visualization tool designed to transform complex smartphone specifications into actionable insights. Built with **Streamlit** and **Plotly**, this dashboard handles real-world data challenges such as missing values and inconsistent formatting to provide a crystal-clear view of the mobile landscape.

## Key Features
- **Market Overview:** Dynamic analysis of brand dominance and quality leadership.
- **Smart Leader Detection:** Intelligent logic that filters out "Missing Data" to identify the true market leaders by volume and rating.
- **Specs Analytics:** An interactive **3D Hardware Cube** visualizing the relationship between RAM, Storage, and Price.
- **Smart Finder & Search Engine:** A robust recommendation engine that filters phones based on budget, hardware specs, and user ratings.
- **Future-Proof Tracking:** Dedicated support for "Coming Soon" models with unannounced pricing.

## Technology Stack
- **Language:** Python 3.9+
- **Framework:** [Streamlit](https://streamlit.io/)
- **Data Manipulation:** Pandas
- **Interactive Graphics:** Plotly Express & Graph Objects
